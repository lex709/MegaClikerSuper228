# MegaClikerSuper228

Clicks everything, click everywhere! Clicks fo real!

## Install

```
pip install -r requirements.txt
```

## Run

Console interface:

```
python super_click.py
```

Gui version:

```
python gui_click.py
```

## Command line options

--delay, -d: delay in seconds for clicks

Example:
```
python super_click.py -d 5
```
